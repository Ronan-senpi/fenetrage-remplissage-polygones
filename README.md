# fenetrage-remplissage-polygones

## Features : 
- Draw polygon (4 points min - White)
- Draw clip window (4 points min - Red)
- Clip (SutherlandHodgman)
- Change fill color
- Scan line filling alogorthim (SLFA)
- Clear viewport
